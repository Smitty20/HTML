Unix Shell Commands

Braden Call

command description usage
cd Change directory cd [direcor-name]
dir list directory contents dir
ls Lists files in current directory ls
alias Create new command in place of another data3
wc Count words, lines, and characters wc [filename]
cat Display File Contents cat [filename]
tail Prints last few lines in a file tail [filename]
sudo allow access to other commands as a different user sudo [command]
find find a direcory or file find [dir-name] -name [file-name]
dig querying DNS nameservers for information about host addresses, mail exchanges, nameservers, and related information dig mt-example.com
nc arbitrary TCP and UDP connections and listens data3
curl transfer data from or to a server curl [options] [URL]
wget "web get", downloads files over a network wget http://website.com/files/file.zip
history Lists commands you've done recently history
which Locate the executable file which -a [filename]
type display the command name's path type [command]
grep Find which files contain a certain word grep [OPTIONS] PATTERN [FILE]
pgrep looks through the currently running processes and lists the process IDs pgrep [options] [process]
pkill sends terminating command to a process pkill [options] [process]
man Online manual /help /about command man [command]
apropos Locate commands by keyword lookup apropos [-dalhvV] [-e|-w|-r] [-s list] [-m system[,...]] [-M path] [-L locale] [-C file] keyword
